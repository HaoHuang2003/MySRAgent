# nd2py 支持的运算符参考

本文档列出 nd2py 符号引擎支持的所有数学运算。

---

## 基本运算

| 运算 | 语法 | 示例 |
|------|------|------|
| 加法 | `+` | `x1 + x2` |
| 减法 | `-` | `x1 - x2` |
| 乘法 | `*` | `x1 * x2` |
| 除法 | `/` | `x1 / x2` |
| 幂运算 | `**` | `x1 ** 2` |
| 负号 | `-` | `-x1` |

**注意**：
- 幂运算使用 `**`，不是 `^`
- `^` 在 Python 中是按位异或

---

## 三角函数

| 函数 | 语法 | 定义域 | 值域 |
|------|------|--------|------|
| 正弦 | `sin(x)` | R | [-1, 1] |
| 余弦 | `cos(x)` | R | [-1, 1] |
| 正切 | `tan(x)` | x ≠ π/2 + nπ | R |
| 反正弦 | `asin(x)` | [-1, 1] | [-π/2, π/2] |
| 反余弦 | `acos(x)` | [-1, 1] | [0, π] |
| 反正切 | `atan(x)` | R | (-π/2, π/2) |

---

## 双曲函数

| 函数 | 语法 | 定义域 | 值域 |
|------|------|--------|------|
| 双曲正弦 | `sinh(x)` | R | R |
| 双曲余弦 | `cosh(x)` | R | [1, ∞) |
| 双曲正切 | `tanh(x)` | R | (-1, 1) |

**应用**：
- `tanh(x)`：S 型激活函数，用于建模饱和现象

---

## 指数和对数

| 函数 | 语法 | 定义域 | 值域 |
|------|------|--------|------|
| 指数函数 | `exp(x)` | R | (0, ∞) |
| 自然对数 | `log(x)` | (0, ∞) | R |
| 常用对数 | `log10(x)` | (0, ∞) | R |
| 以 2 为底 | `log2(x)` | (0, ∞) | R |
| 平方根 | `sqrt(x)` | [0, ∞) | [0, ∞) |

**注意**：
- `log(x)` 是自然对数（以 e 为底）
- 定义域限制：对数函数的输入必须为正

---

## 其他函数

| 函数 | 语法 | 说明 |
|------|------|------|
| 绝对值 | `abs(x)` | |x| |
| Sigmoid | `sigmoid(x)` | 1 / (1 + exp(-x)) |
| 符号函数 | `sign(x)` | x>0 返回 1，x<0 返回 -1，x=0 返回 0 |
| 取整 | `floor(x)` | 向下取整 |
| 取整 | `ceil(x)` | 向上取整 |

## 常量

nd2py 支持以下数学常量：

| 常量 | 值 |
|------|-----|
| π | `pi` ≈ 3.14159... |
| e | `e` ≈ 2.71828... |

**使用示例**：
```
# 圆的面积
area = pi * r ** 2

# 指数衰减
decay = exp(-x / tau)

# 简谐运动
position = A * sin(2 * pi * f * t)
```

---

## 复合函数示例

```python
# 阻尼振荡
damped = exp(-0.1 * x) * sin(2 * pi * x)

# 高斯函数
gaussian = exp(-(x - mu) ** 2 / (2 * sigma ** 2))

# Logistic 增长
logistic = L / (1 + exp(-k * (x - x0)))

# 黑体辐射（普朗克公式）
planck = (2 * h * c ** 2) / (x ** 5 * (exp(h * c / (x * k * T)) - 1))

# 洛伦兹函数
lorentzian = gamma / ((x - x0) ** 2 + gamma ** 2)
```

## 数值稳定性提示

| 问题 | 避免方法 |
|------|----------|
| log(0) 错误 | 使用 `log(x + eps)` |
| 除零错误 | 使用 `x / (y + eps)` |
| 大数溢出 | 使用 `log(exp(x)) = x` 简化 |
| 数值相消 | 使用恒等变换 |

---

## 公式编写最佳实践

1. **使用括号明确优先级**：`(a + b) * c` 而非 `a + b * c`
2. **浮点数用小数点**：`2.0` 而非 `2`
3. **参数用字母表示**：`a`, `b`, `c` 等，让 `fit=True` 优化
4. **避免冗余**：`x * x` 可写为 `x**2`
5. **注意定义域**：确保输入在函数有效范围内
